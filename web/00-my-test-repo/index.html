The Basics of Branches

Welcome to my how-to guide page, we will be exploring the basics of branches!

How to create a branch:

1. Firstly, "cd" into the correct directory and ensure you are connected to your remote repo by typing into the terminal/ command line "git remote v".

2. Type into terminal/ command line "git branch" to view the available branches and your current branch.

3. Type "git branch new branch" or "git checkout -b new branch" to create new branches

4. Type "git status" to view modifications
